js中一些内置对象

js中一些内置对象

Array

Array -length 获取数组的长度(元素个数)

转换数组

  • valueOf() –返回数组对象本身。
  • toString() –把数组转换成字符串。
    • 可以把数组的每一项取出来，拼接成字符串
  • join() –把字符串中的每一项进行拼接
    • 如果不传参数，默认按照逗号拼接 ['张飞','赵云','关羽'].join() –输出结果为 张飞,赵云,关羽
    • 如果传入分隔符，就按照分隔符去分割 ['张飞','赵云','关羽'].join('|')–输出结果为 张飞|赵云|关羽

检测数组

  • arr instanceof Array 返回是布尔值.
  • Array.isArray() es5的新方法、有兼容性

增删方法

  • push() –从后面推入元素，返回新数组的length
    • var arr = [1,2,3]; arr.push(0) // 返回值为 4 arr现在为[1,2,3,0]
    • 参数可以写多个 ，依次添加。
  • pop() –从后面删除元素， 返回被删除的元素
    • var arr = [1, 2, 3]; arr.pop() // 返回值为3 arr现在为[1,2]
  • shift() –从前面删除元素 返回被删除的元素
    • var arr = [1, 2, 3]; arr.shift() // 返回值为1 arr现在为[2,3]
  • unshift() –从前面加入元素 返回新数组的长度
    • var arr = [1,2,3]; arr.unshift(0) // 返回值为 4 arr现在为[0,1,2,3]
    • 参数可以写多个 ，依次添加。

排序方法

  • reverse() 翻转数组
    • 会影响原数组
  • sort()

操作方法

  • concat() – 把参数拼接到当前数组
    • 不会影响原数组

  • slice() –从当前数组中把要的东西复制一份，不影响原来的数组

    • 参数 1开始 2结束 开始的位置能取到 结束的位置取不到 [start,end)
    • 不影响原来的数组
    • 返回值是一个数组，是复制的那个元素

  • splice() –删除或替换当前数组的某些项目

    • 1开始 2删除的个数 3要添加的元素
    • 会影响原数组
    • 返回值是一个数组，是删除或者替换掉的那个元素

位置方法

  • indexOf() –从前往后寻找指定项目在数组中的位置
    • indexof('a')返回的是找到的那个元素的索引
    • indexof('a',1) 第二个参数表示从哪里开始找
    • 如果找不到，就返回-1
  • lastIndexOf() –从后往前寻找指定项目在数组中的位置
    • lastIndexOf('a')返回的是找到的那个元素的索引
    • lastIndexOf('a',-1) 第二个参数表示从哪里开始找
    • 如果找不到，就返回-1

迭代方法

  • every()
  • filter()
  • forEach()
  • map()
  • some()

清空数组的几种方法

  • array.splice(0,arr.length)
    • 删除数组中的所有项目。
  • array.length
    • length 属性可以直接赋值。
  • array = [];
  • 使用delete (不推荐) 因为使用delete 删除数组中的元素，还会占位置，后面的元素的索引并不会发生改变。

String

  • length 属性 可以返回当前字符串中总共有多少字符

字符方法

  • charAt()
    • 返回指定位置处的字符
  • charCodeAt()
    • 获取指定位置处字符的ASCII码
  • str[0]
    • 和charAt() 等效

操作方法

  • concat()
    • 拼接字符串，和+号一样
  • slice()
    • 截取字符串 从start位置开始，截取到end 位置 ，end 取不到 [ )
    • 第一个参数代表开始位置,第二个参数代表结束位置的下一个位置,截取出来的字符串的长度为第二个参数与第一个参数之间的差;若参数值为负数,则将该值加上字符串长度后转为正值;
    • 若第一个参数等于大于第二个参数,则返回空字符串.
  • substring()
    • 截取字符串 从start位置开始，截取到end 位置 ，end 取不到 [ )
    • 第一个参数代表开始位置,第二个参数代表结束位置的下一个位置;若参数值为负数,则将该值转为0;
    • 两个参数中,取较小值作为开始位置,截取出来的字符串的长度为较大值与较小值之间的差.
  • substr()
    • 从start位置开始，截取length个字符
    • 第一个参数代表开始位置,第二个参数代表截取的长度

位置方法

  • indexOf()
    • 从前往后，返回指定内容在元字符串中的位置
  • lastIndexOf()
    • 从后往前， 返回指定内容在元字符串中的位置

匹配方法

  • search()
    • 和indexOf 功能类似，但是可以接收正则
  • replace()
    • 替换元素
    • 第一个参数为将要被替换的元素
    • 第二个参数为要替换成的元素
    • 第一个参数可以接收正则

其他方法

  • trim
    • 去除字符串前后的空白
  • split()
    • 把字符串切割成字符数组
    • split 中的参数为，将按照什么切割

大小写转换方法

  • to(Locale)UpperCase() 转换大写
  • to(Locale)LowerCase() 转换小写

Math

  • Math.PI π
  • Math.min() 最小值
  • Math.max() 最大值
  • Math.ceil() 向上取整
  • Math.floor() 向下取整
  • Math.round() 四舍五入
  • Math.abs() 去绝对值
  • Math.pow(num,power) num的power次幂
  • Math.sin() 正弦
  • Math.cos() 余弦

Date
