js中一些内置对象

Array

Array -length 获取数组的长度(元素个数)

转换数组

  • valueOf() –返回数组对象本身。
  • toString() –把数组转换成字符串。
    • 可以把数组的每一项取出来，拼接成字符串
  • join() –把字符串中的每一项进行拼接
    • 如果不传参数，默认按照逗号拼接 ['张飞','赵云','关羽'].join() –输出结果为 张飞,赵云,关羽
    • 如果传入分隔符，就按照分隔符去分割 ['张飞','赵云','关羽'].join('|')–输出结果为 张飞|赵云|关羽

检测数组

  • arr instanceof Array 返回是布尔值.
  • Array.isArray() es5的新方法、有兼容性

增删方法

  • push() –从后面推入元素，返回新数组的length
    • var arr = [1,2,3]; arr.push(0) // 返回值为 4 arr现在为[1,2,3,0]
    • 参数可以写多个 ，依次添加。
  • pop() –从后面删除元素， 返回被删除的元素
    • var arr = [1, 2, 3]; arr.pop() // 返回值为3 arr现在为[1,2]
  • shift() –从前面删除元素 返回被删除的元素
    • var arr = [1, 2, 3]; arr.shift() // 返回值为1 arr现在为[2,3]
  • unshift() –从前面加入元素 返回新数组的长度
    • var arr = [1,2,3]; arr.unshift(0) // 返回值为 4 arr现在为[0,1,2,3]
    • 参数可以写多个 ，依次添加。

排序方法

  • reverse() 翻转数组
    • 会影响原数组
  • sort()

操作方法

  • concat() – 把参数拼接到当前数组
    • 不会影响原数组

  • slice() –从当前数组中把要的东西复制一份，不影响原来的数组

    • 参数 1开始 2结束 开始的位置能取到 结束的位置取不到 [start,end)
    • 不影响原来的数组
    • 返回值是一个数组，是复制的那个元素

  • splice() –删除或替换当前数组的某些项目

    • 1开始 2删除的个数 3要添加的元素
    • 会影响原数组
    • 返回值是一个数组，是删除或者替换掉的那个元素

位置方法

  • indexOf() –从前往后寻找指定项目在数组中的位置
    • indexof('a')返回的是找到的那个元素的索引
    • indexof('a',1) 第二个参数表示从哪里开始找
    • 如果找不到，就返回-1
  • lastIndexOf() –从后往前寻找指定项目在数组中的位置
    • lastIndexOf('a')返回的是找到的那个元素的索引
    • lastIndexOf('a',-1) 第二个参数表示从哪里开始找
    • 如果找不到，就返回-1

迭代方法

  • every()
  • filter()
  • forEach()
  • map()
  • some()
